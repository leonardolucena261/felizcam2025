# FELIZCAM 2025 - Hotsite Oficial

Hotsite oficial da Festa Literária de Camaçari - FELIZCAM 2025, um evento que celebra a leitura, a arte, a cultura e a identidade local.

## Sobre o Projeto

Este é um hotsite simples e responsivo desenvolvido para divulgar a primeira edição da FELIZCAM, que ocorrerá nos dias 03 e 04 de dezembro de 2025 na Cidade do Saber, em Camaçari. O site apresenta informações sobre o evento, programação, homenageados e como participar como autor.

## Funcionalidades

- Seção sobre o evento e seus objetivos
- Destaque dos guardiões da tradição homenageados
- Programação completa das atividades
- Informações para participação de autores
- Design responsivo para diferentes dispositivos
- Navegação intuitiva e acessível

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (navegação suave)

## Como Utilizar

1. Faça o download dos arquivos do projeto
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Navegue pelas seções do site através do menu ou rolagem da página

## Personalização

O projeto pode ser facilmente personalizado:
- Altere as cores no arquivo CSS modificando as variáveis de cor
- Adicione novas seções seguindo a estrutura existente
- Atualize as informações do evento conforme necessário
